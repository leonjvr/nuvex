## Key Observations

### 1. Self-learning is the runaway #1 demand
`self-improving-agent` (368k) + `Self-Improving + Proactive` (154k) = **522k downloads combined**. This is 3x the next skill. OpenClaw has no native learning/feedback loop — the agent does not improve from its mistakes across sessions. This is the largest user-behaviour gap.

### 2. Structured memory is #2 demand  
`ontology` (157k) fills the gap for persistent, typed, graph-based knowledge. OpenClaw memory is flat files and session compaction — no entity relationships, no typed schema, no query capability.

### 3. Web search requires a skill
Three competing search skills (Prismfy, Baidu, Exa) collectively have ~175k downloads. No native, built-in search exists. Users must install and often compete between options.

### 4. Office document editing is a huge gap (132k)
Word + Excel + PowerPoint from @ivangdavila total 132k downloads. Business users want agents that can draft reports, build spreadsheets, update presentations — none of which are native.

### 5. MCP is being bolted on
Mcporter (53.7k) exists because there is no native MCP client. The MCP ecosystem is the primary way the AI industry is building tool integrations in 2026 — yet OpenClaw requires a community skill to connect to it.

### 6. Free model management is high-demand (54.4k)
"Free Ride" manages free tier LLMs via OpenRouter. Users are cost-constrained and want fallback chains. This is a sign that the onboarding requirement of a paid Anthropic subscription is a barrier.

### 7. Chinese market is large and underserved
Baidu Search (75.1k), Baidu AIPPT (24.3k), Skill Finder Cn (28.4k) are all top-25 skills. The SSRF/network bug (#33086) disproportionately affects Chinese users. No WeChat/WeChat Work channel support.

### 8. Financial data (Polymarket 96k) outperforms many utility skills
Prediction markets / finance data is surprisingly the 4th most downloaded skill, above productivity tools. Signals: speculative/trading users are highly engaged and download-intensive.
